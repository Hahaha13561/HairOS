# HairOS

HairOS, Debian Stable tabanı üzerine inşa edilmiş, hafiflik, hız ve modern estetiği bir araya getirmeyi hedefleyen özel bir Linux dağıtımı projesidir. Hantal ve kaynak tüketen masaüstü ortamları yerine, XFCE'nin hızı ile Pardus'un kullanışlı araçlarını harmanlar.

Bu proje aktif geliştirme aşamasındadır. Arayüz ince ayarları, varsayılan duvar kağıdı gömme işlemleri ve özel sistem logolarının entegrasyon süreci devam etmektedir.

Bundan sonrasını sonra yazarım.
